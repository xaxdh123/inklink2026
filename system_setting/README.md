# 系统设置

## 模块简介
系统设置子程序，提供系统配置与消息中心入口。

## 入口与目录
- 子程序键名：`SystemSetting`
- 配置子目录：`system_setting`
- 当前代码目录：`system_setting/`（与配置名不一致）
- 可执行文件：`system_setting.exe`
- 入口脚本：`system_setting.py`（调用 `system_setting.main()`）

## 打包/运行路径
- 运行：`./bin/system_setting/system_setting.exe`

## 补丁包路径
- `./new_version/system_setting/`（由 COS 对象存储同步）
