# 客服中心

## 模块简介
客服中心子程序，提供客服相关页面与功能入口。

## 入口与目录
- 子程序键名：`CustomerService`
- 子目录：`customer_service`
- 可执行文件：`customer_service.exe`
- 入口脚本：`customer_service.py`（调用 `customer_service.main()`）

## 打包/运行路径
- 运行：`./bin/customer_service/customer_service.exe`

## 补丁包路径
- `./new_version/customer_service/`（由 COS 对象存储同步）
