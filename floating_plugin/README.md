# 浮窗插件

## 模块简介
浮窗插件子程序，提供悬浮窗入口与相关功能。

## 入口与目录
- 子程序键名：`FloatingPlugin`
- 子目录：`floating_plugin`
- 可执行文件：`floating_plugin.exe`
- 入口脚本：`floating_plugin.py`（调用 `floating_plugin.main()`）

## 打包/运行路径
- 运行：`./bin/floating_plugin/floating_plugin.exe`

## 补丁包路径
- `./new_version/floating_plugin/`（由 COS 对象存储同步）
